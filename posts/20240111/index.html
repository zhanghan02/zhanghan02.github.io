ChatGPT账号快速注册

发表于2024-01-11|更新于2024-04-24|我的GAI之路ChatGPT
|字数总计:462|阅读时长:1分钟|阅读量:

AI摘要：本文主要介绍了如何注册并使用 Proton 邮箱和 OpenAI 账户，并申请 API 用于开发工具。首先需要在 Proton 网站注册一个邮箱，选择免费套餐并设置用户名和密码。接着，需要访问 OpenAI 的账户注册页面，填写邮箱地址完成验证。如果需要使用 API，还需要验证手机号并通过接码平台获取验证码进行验证。这样就可以创建 API 并开始开发自己的工具了。

先去Proton网站注册个邮箱

  1. 前往Proton网站，点击右上角的「Create a free account」开始注册邮箱。前往Proton

  2. 选择免费套餐「Proton Free」。

  3. 选择邮箱后缀，设置用户名和密码。后缀推荐选择 proton.me。

  4. 设置您的邮箱账户名称，完成后即可开始使用。

然后再去openai注册账户

  1. 首先打开 OpenAI 的账户注册页面，填入上一步注册的邮箱地址注册。前往OpenAI（这里最好用google浏览器的无痕模式，魔法工具用美国节点最好）

  1. 出现如图页面后，前往您邮箱的收件箱完成验证。

  2. 找到如图的验证邮件，点击「Verify email address」即可完成验证。

  3. 提供您的名、姓及生日，点击继续即可。

搜索openai，选择国家，最好是美国，同时注意需要和你的魔法工具的节点国家一致

然后在openai的api创建界面进行验证，将接码平台收到的验证码输入完成验证，就可以创建API了。
